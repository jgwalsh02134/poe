# Poe Containment Search Map

Interactive map for the Poe search containment perimeter (Menands / Albany area).

**Point Last Seen:** 162 Menands Road / Orchard Grove • Sunday, July 12, ~9 PM

## Live Map (GitHub Pages)

Once GitHub Pages is enabled, the map will be available at:

**https://jgwalsh02134.github.io/poe/**

## How to Enable GitHub Pages

1. Go to your repository: https://github.com/jgwalsh02134/poe
2. Click **Settings** (top right)
3. In the left sidebar, click **Pages**
4. Under "Build and deployment":
   - **Source**: Select **Deploy from a branch**
   - **Branch**: `main`
   - **Folder**: `/` (root)
5. Click **Save**

GitHub will build and deploy the site. It usually takes 1–2 minutes.

Your live site will be at: **https://jgwalsh02134.github.io/poe/**

## Files

- `index.html` — The interactive Leaflet map (self-contained)
- `Poe_Containment_Search_Map.html` — Original standalone version
- `Poe_Containment_Search_Map.geojson` — GeoJSON data for GIS tools

## Usage

- Open the map in any modern browser.
- All zones, containment posts (P1–P9), travel corridors, and the outer perimeter are shown.
- Use the sidebar for operational guidance.
- Click markers and polygons for details.

## Mobile Friendly

The layout adapts on smaller screens (sidebar moves below the map).
